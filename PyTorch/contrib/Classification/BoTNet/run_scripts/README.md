## 参数介绍

参数名 | 说明 | 示例
-----------------|-----------------|-----------------
img_size |图像尺寸 | --img_size 224
batch_size | 批次大小 | --batch_size 64
lr | 学习率 | --lr 0.0002
epochs | 训练轮次 | --epochs 100
output | 模型保存路径 | --output "../my_saved_models/attention/"
num_heads | 注意力层header数 | -num_heads 4
name | 模型名 | ----name ""
